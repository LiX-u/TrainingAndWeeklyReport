int GCD(int a, int b)
{

## 凸包
```cpp
void Graham()
{
    int k = 0;
    for(int i = 1; i < n; i++)
        if(p[i].y < p[k].y || (p[i].y == p[k].y && p[i].x < p[k].x))
            k = i;
	
    swap(p[k], p[0]);
    sort(p+1, p+n, cmp);
    s[0] = p[0];
    s[1] = p[1];
    top = 1;

    for(int i = 2; i < n; i++)
    {
        while(top >= 1 && chaji(s[top-1], s[top], p[i]) >= 0)
            top--;
        top++;
        s[top] = p[i];
    }
}
```
叉积与极角排序都是为凸包所服务的<br>
主要思想就是将所有的点都用一个多边形包括<br>
以解决任意两点最长距离、最外点连多边形周长等<br>
要注意的就是点的数目是1和2的情况要判断下<br>
