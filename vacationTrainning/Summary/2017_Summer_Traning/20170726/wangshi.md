#  凸包
~~~cpp
  凸集：任意两点的连线都在这个集合内的集合
  凸包：包含集合Ｓ的所有凸集的交集就是集合Ｓ的凸包。
  叉积：ａｘｂ>0　ａ在ｂ的顺时针
     　ａｘｂ<0　ａ在ｂ的逆时针
      写凸包时最好只固定用一种，否则容易混乱ＱＡＱ
  卷包裹法：对所有点遍历，ＮＨ
  graham算法:维护一个凸壳，去除所有影响凸度的点，剩下的点便是凸包点
            左转（ａ逆时针到ｂ）则入栈，右转（ａ顺时针到ｂ）则ａ出栈，再从栈
            中找出两个点与第三个点进行判断，直到左转或者栈中元素小于２（Ｎｌｏｇ２Ｎ）
~~~
